Politique de Confidentialité — MedDevice Learn

Dernière mise à jour : Juin 2026

1. Présentation

MedDevice Learn est une application éducative dédiée aux professionnels et étudiants en santé. Elle ne collecte aucune donnée personnelle.

2. Données collectées

Aucune donnée personnelle n'est collectée. L'application fonctionne entièrement hors ligne.

3. Stockage des données

Toutes les données (progression, résultats de quiz) sont stockées uniquement sur votre appareil local.

4. Partage des données

Aucune donnée n'est partagée avec des tiers.

5. Contact

Pour toute question : meddevicelearn@gmail.com
